Parágrafos HTML

Um parágrafo sempre começa em uma nova linha e geralmente é um bloco de texto

Exibição HTML

Este parágrafo contém muitas linhas no código-fonte, mas o navegador as ignora.

Este parágrafo contém muitos espaços no código-fonte, mas o navegador os ignora.

O número de linhas em um parágrafo depende do tamanho da janela do navegador. Se você redimensionar a janela do navegador, o número de linhas neste parágrafo mudará.

Quebras de linha HTML

O <br> elemento HTML define uma quebra de linha.

O Problema do Poema

Este poema será exibido em uma única linha:

Minha Bonnie está sobre o oceano. Minha Bonnie está sobre o mar. Minha Bonnie está sobre o oceano. Oh, traga minha Bonnie de volta para mim.

Solução - O elemento HTML <pre>

O elemento HTML <pre> define texto pré-formatado.

        Minha Bonnie está sobre o oceano.

        Minha Bonnie está sobre o mar.

        Minha Bonnie está sobre o oceano. 

        Oh, traga minha Bonnie de volta para mim.
