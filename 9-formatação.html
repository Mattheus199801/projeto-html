Formatação HTML

O HTML contém vários elementos para definir texto com um significado especial.

Elementos de formatação HTML

Os elementos de formatação foram projetados para exibir tipos especiais de texto:

  • <strong> - Texto importante
  • <em> - Texto enfatizado
  • <mark> - Texto marcado
  • <small> - Texto menor
  • <del> - Texto excluído
  • <ins> - Texto inserido
  • <sub> - Texto subscrito
  • <sup> - Texto sobrescrito

Elementos HTML <strong>

O elemento HTML define o texto com grande importância. O conteúdo interno é normalmente exibido em negrito. <strong>

Este texto é importante!

Elementos HTML <em>

O elemento HTML define texto enfatizado. O conteúdo interno é normalmente exibido em itálico. <em>

Ponta: Um leitor de tela pronunciará as palavras com ênfase, usando ênfase verbal. <em>

Este texto é enfatizado.

Elemento HTML <small>

O elemento HTML define Texto menor: <small>

Este é um texto menor.

Elemento HTML <mark>

O elemento HTML define o texto que devem ser marcados ou destacados: <mark>

Não se esqueça de comprar leite hoje.

Elemento HTML <del>

O elemento HTML define o texto que foi excluído de um documento. Os navegadores geralmente atingem uma linha através de texto excluído: <del>

Minha cor favorita é azul vermelho.

Elemento HTML <ins>

O elemento HTML define um texto que foi inserido em um documento. Os navegadores geralmente sublinham inserido Texto: <ins>

Minha cor favorita é azul vermelho.

Elemento HTML <sub>

O elemento HTML define texto subscrito. O texto subscrito aparece meio caractere abaixo da linha normal, e às vezes é renderizado em uma fonte menor. O texto subscrito pode ser usado para fórmulas químicas, como H2<sub>O

Este é um texto subscrito.

Elemento HTML <sup>

O elemento HTML define texto sobrescrito. O texto sobrescrito aparece meio caractere acima do normal linha, e às vezes é renderizado em uma fonte menor. Texto sobrescrito pode ser usado para notas de rodapé, como WWW<sup>[1]

Este é um texto sobrescrito.
